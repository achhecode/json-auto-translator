# JSON Auto Translator

Automatically translate JSON files to multiple languages using browser automation.

## Quick Start

1. **Install** dependencies:

```bash
pip install -r requirements.txt
```

2. Place your JSON file in the project folder
3. Run the translator:

Configure environment:

Copy .env.example to .env:

```bash
cp .env.example .env
```

Edit .env to set your file paths (or leave as default)

## Run

```bash
python main.py
```

That's it! Translated files will be saved in the translations/ folder.
